# Smart Child Safety & Tracking System
### Texas Instruments WiSH (Women in Semiconductors and Hardware) Capstone Project

## Overview
This project was developed as the capstone project of the **Texas Instruments Women in Semiconductors and Hardware (WiSH)** mentorship program. It proposes a low-cost, energy-efficient child safety system designed to improve child security during daily commutes, particularly in rural and semi-urban regions.

The solution combines embedded system design, intelligent decision-making, and communication technologies to automatically detect abnormal situations and notify parents without requiring continuous user interaction.

---

## Problem Statement
Children are vulnerable to getting lost, kidnapping, or distress during daily travel. Existing tracking solutions often suffer from high costs, limited battery life, poor network availability, or dependence on manual SOS activation.

This project aims to design an affordable and scalable embedded solution capable of automatically detecting emergencies and providing timely alerts.

---

## Key Features

- Geofencing-based safe zone monitoring
- Automatic route anomaly detection
- SOS emergency alert mechanism
- GNSS-assisted location tracking
- Low battery detection
- Energy-efficient power management
- Parent monitoring interface
- Cost-effective embedded system architecture

---

## System Architecture

The proposed solution consists of multiple functional layers:

- Power Management Layer
- Location & Communication Layer
- Decision Layer
- Parent Monitoring Interface

The decision layer continuously evaluates:

- Safe zone transitions
- Route deviations
- SOS events
- Low battery conditions

and generates alerts whenever abnormal behaviour is detected.
